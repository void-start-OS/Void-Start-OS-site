VSOAPI とは？

VSOAPI（Void-Start-OS Application Programming Interface）は、 Void-Start-OS のユーザーランド向けに提供される公式 API セットです。 コンソール、メモリ、時間、システム情報など、 OS の基本機能へアクセスするための統一されたインターフェイスを提供します。

提供される API

  • Console API — 文字出力・入力
  • Memory API — メモリ確保・解放
  • Time API — Sleep / TickCount
  • System API — システム情報取得
  • File API（予定）
  • Process API（予定）
